## ğŸš€ Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd timmer
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Add Images (Optional)**
   - Place your storefront image as `public/securenexa-hero.jpg`
   - Place your office/showroom image as `public/securenexa-office.jpg`
   - Or update image paths in `src/components/Hero.jsx` if using different filenames

4. **Start development server**
   ```bash
   npm run dev
   ```
   The site will be available at `https://github.com/encrypt2004/securenexa-cctv-website`

5. **Build for production**
   ```bash
   npm run build
   ```
   Output will be in the `dist/` directory
